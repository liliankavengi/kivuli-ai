# Kivuli AI 
**Empowering Kenyan SMEs with AI-driven Financial Trust & SDG-aligned Mentorship.**

Kivuli AI is a full-stack fintech platform that bridges the gap between informal M-Pesa transaction data and formal credit readiness. Using **Gemini 3 Flash**, it analyzes business health and provides actionable advice aligned with **UN Sustainable Development Goal 8 (Decent Work and Economic Growth)**.

---

## Features
- **AI Trust Scoring:** Converts raw M-Pesa transaction history into a bankable trust score.
- **SDG 8 Mentor:** Real-time financial coaching powered by Google Gemini.
- **Automated Bookkeeping:** Seamless integration with Safaricom Daraja API (Sandbox).
- **Fintech Dashboard:** Responsive React UI with data visualization.

## Tech Stack
- **Frontend:** React 19, Vite, Tailwind CSS, Recharts, Lucide Icons.
- **Backend:** Django 6.0, Django REST Framework.
- **AI:** Google Generative AI (Gemini 3 Flash SDK).
- **Database:** SQLite (Development) / PostgreSQL (Production).
